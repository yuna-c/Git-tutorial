Git

  • Git 설치
  • Git 설치(Mac)
  • cd desktop
  • mkdir 생성폴더명
        
$ git init : 빈 레퍼지토리 생성
$ git add . : 변경 사항 기록
$ git status : 프로젝트 어떻게 관리 되고 있는지 확인
$ git config --global user.name '사용자명'
$ git config --global user.email '이메일@주소'
$ git config user.name 확인
$ git config user.email 확인
$ git remote add origin https://github.com/yuna-c/Git-tutorial.git 저장소 등록
$ git commit -m "분기점" 등록
$ git log 아이디값 확인
clear 터미널 초기화
$git checkout 원하는 커밋 시점 돌아가기
$git checkout master 최신커밋 돌아가기
$git branch 브랜치명 가지뻗기
$git branck -v 브랜치 버전
